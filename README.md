# Portfolio & Blog

Personal portfolio and blog for Kyoung Whan Choe.

**Live site:** https://kywch.github.io

## Tech Stack

- **Jekyll** (static site generator)
- **GitHub Pages** (hosting)
- **Theme:** Modified [jekyll-theme-minimal](https://github.com/pages-themes/minimal)

## Features

- Blog with client-side pagination
- Project showcase
- Responsive navigation
- Google Analytics integration

## Local Development

```bash
bundle install
bundle exec jekyll serve
# Visit http://localhost:4000
```
